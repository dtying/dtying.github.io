Build my very first News website - 0

2016-12-22 • tags: React, Redux

Learning the basics of React and Redux, I had recently built up my very first web application - News. It’s a simple web page where you can always scroll down for news headlines, and click in to read in details. The news API I used is The guardian, which is a free open source.

Adopted the responsive web design, the website displays in single column from mobile devices, 2-column from tablets and 3-column from desktop. As it’s a small project, I used one single global css file with mobile-first pattern.

In the next few posts, I’m going to share my experiences and some good tutorials which I find very useful. I hope that can help the beginners just like me who are new to Redux and React.

Redux

I highly recommend the video course Getting Started with Redux narrated by Dan Abramov, author of Redux. The course is really easy to follow, and it covers almost every important concept. It not only tells you how to use Redux, but also illustrates how Redux was built.
It’s followed by advanced course Building React Applications with Idiomatic Redux, which teaches you the best practices to use Redux together with React.
With the basic knowledge of Redux, you can continue by reading the official documentation, or getting your hands dirty in playing around with the demo projects.

React

Compare with Redux, React seems easier to pick up. My suggestion is to build a webpage using React only, and then try to rebuild the same with Redux.

What’s next

In the next post, I’ll discuss about how to build up the website. It won’t be a step-by-step handbook, but a bit like a reflection at finishing a project.
